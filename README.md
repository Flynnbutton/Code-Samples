# Code-Samples
Samples of code developed for research projects and coursework from my graduate studies.


Code Samples — Flynn Button
This repository contains three code samples submitted as part of job applications for quantitative research and data analysis roles. The scripts represent work completed during graduate studies at the Barcelona School of Economics and demonstrate proficiency in R and Stata across a range of statistical and econometric methods.

--------------------------------------------------------------------------------------------------------------------------------------

Files
Flynn_Button_Matching_R.R
Propensity score matching analysis evaluating the impact of an internet infrastructure programme on labour market outcomes in Peru, using ENAHO household survey data. Includes data preparation, matching algorithm implementation, balance checking, and outcome analysis. Developed as part of a collaborative graduate coursework project.


Flynn_Button_ML_R.R
Machine learning analysis using 2021 Canadian Census microdata to predict low-income status in Nova Scotia. Implements and compares three classifiers — Random Forest, XGBoost, and Elastic Net logistic regression — with hyperparameter tuning via cross-validation and model evaluation using confusion matrices and ROC curves. Developed as part of a collaborative graduate coursework project.


Flynn_Button_Stata_Sample.do
Excerpt from a panel data analysis examining the causal impact of public library access on secondary school attainment in England. Covers data cleaning and standardisation, treatment variable construction using geographic proximity measures, and high-dimensional fixed effects regression with school disadvantage interaction terms. Developed as part of a collaborative Master's thesis research project at the Barcelona School of Economics.
